# Slack 채널 구조

> 1인 워크스페이스 기준. 봇과의 대화는 DM으로 수행.

## 채널 목록

| 채널 | 용도 | 빈도 |
|------|------|------|
| `#lab-report` | 시스템 주간 리포트 | 매주 월 09:00 KST |
| `#lab-papers` | 논문 수집·리뷰 리포트 | 데일리 + 주간 |
| `#lab-project` | 프로젝트 진척·마감·액션 | 주간 + 이벤트 |
| `#lab-alerts` | 시스템 알림 | 즉시 (이벤트 발생 시) |

## 채널 상세

### `#lab-report` — 시스템 주간 리포트
- **발신**: 매주 월요일 09:00 KST
- **내용**:
  - 7일 에이전트 사용 추이 (일별 바 차트)
  - 에이전트 사용 순위 (메달)
  - 대화 모드 분포 (normal / debate / pipeline)
  - 파이프라인 실행 이력 + 성공률
  - 미활용 에이전트/모드 권장사항

### `#lab-papers` — 논문 리포트
**데일리** (매일 09:00 KST):
- Scout 전일 수집 논문 목록 (제목, 저널, 관련도 점수)
- 상위 5편 핵심 요약 (1-2문장)
- 키워드 태그

**주간** (매주 월 09:00 KST):
- 주간 수집 논문 통계 (총 수, 소스별)
- 상위 5편 상세 요약
- 주간 연구 키워드 트렌드
- 관심 분야 커버리지 분석
- (추후) literature agent 호출 기반 분석 질문 요약

### `#lab-project` — 프로젝트 리포트
**주간** (매주 월 09:00 KST):
- 등록된 프로젝트별 진척률 (진행 바)
- 다음 2주 마감 알림 (D-day 색상 표시)
- 완료된 마일스톤 요약
- 다음 주 예정 작업 / 액션 아이템
- 미완료 항목 리마인더

**이벤트 트리거**:
- 마감 D-3 이내 긴급 리마인더
- 마일스톤 완료 알림

### `#lab-alerts` — 시스템 알림
- **에이전트 오류**: 호출 실패, timeout, 연속 실패 감지
- **서비스 장애**: 오케스트레이터 / Slack Bot / Qdrant 다운
- **파이프라인 실패**: 단계별 실패 지점 포함
- **Scout 수집 오류**: API 한도 초과, 소스 접근 불가
- **리소스 경고**: DB 용량 초과, 디스크 부족
- **모델 API 오류**: OpenAI/Anthropic/Google 키 만료, rate limit
- **스케줄 미실행**: 리포트 생성 실패 알림
